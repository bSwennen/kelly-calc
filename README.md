# Kelly Calculator

Vue app to compute the optimal bet based on the [Kelly criterion](https://en.wikipedia.org/wiki/Kelly_criterion)

## Project setup

```
# yarn
yarn

# npm
npm install

# pnpm
pnpm install
```

### Compiles and hot-reloads for development

```
# yarn
yarn dev

# npm
npm run dev

# pnpm
pnpm dev
```

### Compiles and minifies for production

```
# yarn
yarn build

# npm
npm run build

# pnpm
pnpm build
```
